# Borealise Scripts

A collection of userscripts for **Borealise**, designed to improve your experience on the platform.

## Available Scripts

### ❤️ Borealise Auto Woot

Automatically upvotes every new track as it starts playing.

### Features

- ✅ Automatically upvotes every song
- ✅ Doesn't click if you've already voted
- ✅ Lightweight and fast
- ✅ Compatible with Tampermonkey & Violentmonkey

## Installation

### 1. Install a userscript manager

- [Tampermonkey](https://www.tampermonkey.net/)
- [Violentmonkey](https://violentmonkey.github.io/)

### 2. Install the script

👉 **[Click here to install Borealise Auto Woot](https://github.com/zealest/borealise-scripts/raw/main/borealise-auto-woot.user.js)**

Your userscript manager should automatically detect the script and prompt you to install it.

## Usage

1. Open https://app.borealise.com/
2. Join any room.
3. The script will automatically upvote each new song.
